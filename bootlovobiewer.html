What is it?

It's an Android application for OnePlus One that help you to manage the logo image on OnePlus One (bacon) device.

What will I need?

You'll need

  • Rooted device. If you don't know how to do go here: How to Root the OnePlus One and Unlock Bootloader and Install Custom Recovery – XDA Developer TV
  • Enable Unknown Source in Settings
  • Download the APK file from the post on XDA-DEV ( take a look on the left! )

How to use it?

It's simple!

Open the application, give it the root permissions and reload by pressing . Now you have the bootlogo image loaded. Long press on the image let you change the selected image. Keep in mind that the width and the heigh are shown on top of the image. The color are RGB without Alpha channel!
